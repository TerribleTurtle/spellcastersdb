# SpellcastersDB

A modern, high-performance database and deck builder for Spellcasters. Built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Scripts

- `npm run dev`: Runs the development server.
- `npm run build`: Builds the application for production.
- `npm run check-data`: Verifies connection to the Spellcasters Community API.
- `npm run type-check`: Runs TypeScript compiler check.
