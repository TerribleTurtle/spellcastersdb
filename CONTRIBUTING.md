# Contributing to SpellcastersDB

Thank you for your interest in contributing! This project is a community-driven database and builder for Spellcasters.

## Getting Started

1.  **Clone the Repo**: `git clone https://github.com/TerribleTurtle/spellcastersdb.git`
2.  **Install Dependencies**: `npm install`
3.  **Run Development Server**: `npm run dev`
4.  **Check Data**: `npm run check-data` to verify the API connection.

## Development Workflow

- Create a branch for your feature or fix.
- Ensure types pass with `npm run type-check`.
- Submit a Pull Request with a clear description of changes.
